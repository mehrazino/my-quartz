## تنظیم راست‌چین متون فارسی به صورت خودکار

**توضیح:** این تغییر باعث می‌شود که متون فارسی و عربی به صورت خودکار راست‌چین شوند، در حالی که کدها و فرمول‌های ریاضی چپ‌چین باقی می‌مانند.

**کد:**

```css
/* فایل: quartz/styles/custom.scss */

/* Auto RTL direction for Persian/Arabic text */
p, li, blockquote p, h1, h2, h3, h4, h5, h6, td, th, div, span {
  &:not(pre *, code, .katex *, .math *) {
    &:lang(ar), &:lang(fa) {
      direction: rtl;
    }
    
    &:not([lang]) {
      unicode-bidi: plaintext;
    }
  }
}

/* Ensure code blocks and math expressions always maintain LTR direction */
pre, code, .katex, .math {
  direction: ltr !important;
  text-align: left !important;
}
```

**روش پیاده‌سازی دستی:**
1. فایل `quartz/styles/custom.scss` را باز کنید
2. کد CSS بالا را به انتهای فایل اضافه کنید
3. با اجرای دستور `npm run build` تغییرات را اعمال کنید

## تنظیم فونت وزیرمتن برای دیاگرام‌ها

**توضیح:** این تغییر باعث می‌شود که متون در دیاگرام‌های Mermaid از فونت وزیرمتن استفاده کنند که برای نمایش متون فارسی مناسب‌تر است.

**کد:**

```css
/* فایل: quartz/styles/custom.scss */

/* Set Vazirmatn font for Mermaid diagrams */
.mermaid text, .mermaid .label {
  font-family: Vazirmatn !important;
}

/* Ensure Vazirmatn is used in all SVG text in diagrams */
svg text {
  font-family: Vazirmatn !important;
}
```

```typescript
/* فایل: quartz/components/scripts/mermaid.inline.ts */

mermaid.initialize({
  startOnLoad: false,
  securityLevel: "loose",
  theme: darkMode ? "dark" : "base",
  themeVariables: {
    fontFamily: "Vazirmatn, " + computedStyleMap["--codeFont"],
    primaryColor: computedStyleMap["--light"],
    primaryTextColor: computedStyleMap["--darkgray"],
    primaryBorderColor: computedStyleMap["--tertiary"],
    lineColor: computedStyleMap["--darkgray"],
    secondaryColor: computedStyleMap["--secondary"],
    tertiaryColor: computedStyleMap["--tertiary"],
    clusterBkg: computedStyleMap["--light"],
    edgeLabelBackground: computedStyleMap["--highlight"],
  },
})
```

**روش پیاده‌سازی دستی:**
1. فایل `quartz/styles/custom.scss` را باز کنید و کد CSS بالا را به انتهای آن اضافه کنید
2. فایل `quartz/components/scripts/mermaid.inline.ts` را باز کنید
3. قسمت `fontFamily` در تنظیمات را به صورت `fontFamily: "Vazirmatn, " + computedStyleMap["--codeFont"],` تغییر دهید
4. با اجرای دستور `npm run build` تغییرات را اعمال کنید

## غیرفعال کردن اثر هاور برای کارت‌ها در نمای موبایل

**توضیح:** این تغییر باعث می‌شود که اثر هاور (بلند شدن) کارت‌های منو در نمای موبایل غیرفعال شود تا تجربه کاربری بهتری ایجاد شود.

**کد:**

```css
/* فایل: quartz/components/MenuCards.tsx */

/* اضافه کردن این بخش به CSS کامپوننت */
@media (max-width: 600px) {
  .menu-card:hover {
    transform: none;
    box-shadow: none;
    background-color: var(--lightgray);
  }
  
  .menu-card:hover svg {
    stroke: var(--darkgray);
  }
  
  .menu-card:hover span {
    color: var(--darkgray);
  }
}
```

**روش پیاده‌سازی دستی:**
1. فایل `quartz/components/MenuCards.tsx` را باز کنید
2. بخش CSS کامپوننت را پیدا کنید (معمولاً در متغیر `MenuCards.css` قرار دارد)
3. کد CSS بالا را به انتهای بخش CSS اضافه کنید
4. با اجرای دستور `npm run build` تغییرات را اعمال کنید

## نتیجه‌گیری

با اعمال این تغییرات، قالب سایت بهینه‌سازی شده تا با محتوای فارسی سازگارتر باشد و تجربه کاربری بهتری ارائه دهد. این تغییرات شامل بهبود در جهت متن، فونت‌ها، چیدمان عناصر، و قابلیت‌های جدید مانند دکمه‌های شبکه‌های اجتماعی می‌شود.

برای اعمال هر یک از این تغییرات، پس از اصلاح فایل‌های مربوطه، باید پروژه را با دستور `npm run build` بازسازی کنید تا تغییرات در سایت نهایی اعمال شوند.
